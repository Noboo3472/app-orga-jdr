# Planificateur de disponibilité des jeux

Application web permettant de centraliser les disponibilités d’un groupe de joueurs, de calculer automatiquement la première date commune avec celle du MJ, puis d’envoyer le résultat sur Discord.

## Fonctionnalités

- Saisie des disponibilités des joueurs.
- Prise en compte des disponibilités du MJ.
- Calcul automatique de la première date commune.
- Envoi du résultat sur Discord.
